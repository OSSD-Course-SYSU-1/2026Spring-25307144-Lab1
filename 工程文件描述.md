# 鸿蒙基础工程文件解析文档
## 1. 项目基础信息
1. 项目来源：OpenHarmony官方applications_app_samples开源仓库
2. 项目名称：ArkTSClock简易时钟
3. 开发模型：Stage模型
4. 开发语言：ArkTS
5. 运行环境：DevEco Studio + OpenHarmony模拟器

## 2. 项目整体目录结构
ArkTSClock/
├── entry/ # 主应用模块
│ └── src/main
│ ├── ets/pages/ # 页面代码，Index.ets 为主首页时钟界面
│ ├── resources/base/ # 文字、样式资源
│ └── module.json5 # 页面路由配置文件
├── oh-package.json5 # 项目依赖配置
├── build-profile.json5 # 编译构建配置
└── hvigorfile.ts # 构建脚本

## 3. 核心代码文件功能说明
1. entry/src/main/ets/pages/Index.ets
时钟首页入口，实现实时时分秒动态刷新、页面布局、时间渲染逻辑。
2. module.json5
配置应用默认启动页面，注册页面路由信息。
3. resources资源目录
统一存放文本字符串、尺寸样式等静态资源。
4. oh-package.json5
记录项目版本、第三方依赖包信息。

## 4. 运行说明
使用DevEco Studio内置鸿蒙模拟器，导入该工程即可直接编译运行。
