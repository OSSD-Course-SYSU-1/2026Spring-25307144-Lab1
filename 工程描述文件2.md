# Icon Main Color 项目详细描述文档

## 📋 项目概述

**项目名称**：icon-main-color2  
**Bundle ID**：com.example.iconmaincolor  
**版本**：1.0.0 (versionCode: 1000000)  
**项目类型**：HarmonyOS Feature Module  
**开发语言**：ArkTS  

### 核心功能
本项目实现了一个**根据图标自适应背景色**的HarmonyOS应用示例。通过图像处理算法，自动提取图标的主要颜色并生成和谐的背景色，同时支持深色/浅色主题切换。

---

## 🎯 功能特性

### 1. 智能颜色提取
- **图像解码**：将图标解码为PixelMap对象（40×40分辨率）
- **像素分析**：遍历所有像素值，统计出现频率
- **主色识别**：找出出现次数最多的像素作为主要颜色
- **颜色转换**：RGB ↔ HSV 颜色空间转换

### 2. 自适应背景色生成
- **浅色主题**：高明度、低饱和度的背景色
- **深色主题**：低明度、高饱和度的背景色
- **动态调整**：根据色相(Hue)分段调整饱和度和明度

### 3. 主题管理系统
- **单例模式**：ThemeManager统一管理主题状态
- **双向切换**：支持浅色/深色主题实时切换
- **状态同步**：使用@Watch监听主题变化，自动更新UI

### 4. 响应式UI设计
- **多设备适配**：支持直板机、平板、2in1设备
- **沉浸式窗口**：集成WindowUtil实现沉浸式体验
- **动态布局**：根据设备类型调整UI布局参数

## 🔧 核心技术实现

### 1. 图像处理流程 (ImageUtils.ets)

```typescript
// 核心流程
async function updateIconBackgroundColors() {
  1. image2PixelMap(icon)           // 图像转PixelMap (40×40)
  2. traverseAllPixelValue(pixelMap) // 提取所有像素值
  3. findMaxPixelValue(pixels)       // 找出主要颜色
  4. numberToRgb(maxPixel)           // 整数转RGB
  5. rgb2hsv(rgb)                    // RGB转HSV
  6. modifySVValue(hsv, isDarkTheme) // 调整饱和度和明度
  7. hsv2rgb(hsv)                    // HSV转RGB
  8. rgbToNumber(rgb)                // RGB转整数颜色值
}
```

**关键优化**：
- 分辨率设为40×40，平衡性能与准确性
- 使用整数存储像素值，提升计算效率
- 跳过透明像素(Alpha=0)，避免干扰

### 2. 颜色空间转换 (ColorUtils.ets)

**RGB → HSV 转换**：
```typescript
// 归一化RGB值
red = color.red / 255
green = color.green / 255
blue = color.blue / 255

// 计算HSV
max = Math.max(red, green, blue)
min = Math.min(red, green, blue)
delta = max - min

hue = 根据最大值分量计算色相
saturation = (max === 0) ? 0 : delta / max
value = max
```

**HSV → RGB 转换**：
```typescript
// 分段计算RGB
h60 = hue / 60
hi = h60 % 6  // 色相区间
f = h60 - Math.floor(h60)

// 根据hi值选择不同的计算公式
p = value * (1 - saturation)
q = value * (1 - f * saturation)
t = value * (1 - (1 - f) * saturation)
```

### 3. 主题自适应算法 (ImageUtils.ets)

**浅色主题**：
```typescript
// 根据色相分段调整
if (hue > 0 && hue <= 60) {
  saturation = 0.12  // 低饱和度
  value = 0.93       // 高明度
} else if (hue > 60 && hue <= 190) {
  saturation = 0.08
  value = 0.91
} else if (hue > 190 && hue <= 270) {
  saturation = 0.10
  value = 0.93
} else {
  saturation = 0.08
  value = 0.93
}
```

**深色主题**：
```typescript
// 深色主题使用相反策略
if (hue > 0 && hue <= 60) {
  saturation = 0.30  // 较高饱和度
  value = 0.20       // 低明度
} else if (hue > 60 && hue <= 190) {
  saturation = 0.25
  value = 0.18
} else if (hue > 190 && hue <= 270) {
  saturation = 0.35
  value = 0.22
} else {
  saturation = 0.25
  value = 0.20
}
```

### 4. 状态管理 (Index.ets)

```typescript
@Entry
@Component
struct Index {
  @State isDarkTheme: boolean = false
  @State @Watch('onThemeChanged') isDarkTheme: boolean = false
  
  // 主题变化监听器
  onThemeChanged() {
    this.updateIconBackgroundColors()
  }
  
  // 生命周期
  async aboutToAppear() {
    this.isDarkTheme = this.themeManager.isDarkTheme()
    await this.updateIconBackgroundColors()
  }
}
```
